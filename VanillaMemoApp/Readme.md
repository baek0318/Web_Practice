# 토이 프로젝트

## 주제
노트 앱(memozi)

## 요구사항
- **로그인**
- **노트 기능**<br />
노트를 작성할수있다 (markdown도 지원)<br />
       - 날짜도 생성(수정시 업데이트)  
       - 제목 작성  
       - 내용 작성  
       - 이미지 첨부  
폴더를 생성해서 정리 가능(쉽게 할 수 있는 방법 생각하기) (배경색이나 다른 이모지로 구분하기)<br />
타이핑이 끝나고 30초뒤에 자동 저장?<br />
공유가 가능하다?<br />
- **추후 업데이트 요구사항**<br />
제목을 분석해서 자동 분류기능<br />
태그 분류가 가능<br />
       - 투두 리스트 기능<br />
할 목록 작성이 가능하다<br />
목록 작성시에 시간 날짜 설정가능??(이건 잘 모르겠음)<br />
